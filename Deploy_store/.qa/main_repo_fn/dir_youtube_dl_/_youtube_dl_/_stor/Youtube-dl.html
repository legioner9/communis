Установите инструмент Youtube-dl

Официально рекомендуемый метод установки Youtube-dl очень прост: просто сохраните его в PATH в соответствии с платформой операционной системы, которую вы используете, вы можете запустить его и сразу же начать использовать:

sudo curl -L https://yt-dl.org/downloads/latest/youtube-dl -o /usr/local/bin/youtube-dl

Если в вашей системе нет curl, вы можете использовать вместо него wget:

sudo wget https://yt-dl.org/downloads/latest/youtube-dl -O /usr/local/bin/youtube-dl

После завершения загрузки необходимо указать разрешения на ее выполнение:

sudo chmod a+rx /usr/local/bin/youtube-dl

В качестве альтернативы вы также можете использовать pip для его установки:

sudo -H pip install --upgrade youtube-dl

Помимо загрузки напрямую с официального сайта, многие распространенные дистрибутивы LinuxОфициальный репозиторий или какой-нибудь PPAYoutube-dl включен в оба, например:

  • Arch Linux sudo pacman -S youtube-dl
  • Debian или Ubuntu sudo apt install youtube-dl
  • Fedora sudo dnf install youtube-dl

Youtube-dl подробные уроки и примеры для начинающих

1. Скачайте видео или плейлисты.

Чтобы загрузить видео или весь список воспроизведения видео с Youtube, просто используйте URL-адрес напрямую:

youtube-dl https://www.youtube.com/watch?v=iJvr0VPsn-s

Если вы хотите указать название видео после загрузки, вы можете использовать следующий метод:

youtube-dl -o 'A REAL Back to School Laptop Guide.mp4' https://www.youtube.com/watch?v=iJvr0VPsn-s

Конечно, вы также можете приложить более подробную информацию при загрузке видео. Доступные параметры: заголовок, имя загрузчика (название канала), дата загрузки видео и т. Д .:

youtube-dl -o '%(title)s by %(uploader)s on %(upload_date)s in %(playlist)s.%(ext)s' https://www.youtube.com/watch?v=iJvr0VPsn-s

2. Загрузите несколько видео.

Иногда нам нужно загрузить несколько разных видео с Youtube одновременно. В настоящее время нам нужно только разделить несколько URL-адресов пробелами:

youtube-dl <url1> <url2>

Или вы можете поместить все URL-адреса видео, которые хотите загрузить, в текстовый файл и передать их в качестве параметра Youtube-dl:

youtube-dl -a url.txt

Приведенная выше команда загрузит все видео, на которые указывает URL-адрес в файле url.txt.

3. Скачивайте только аудио (в видео)

Youtube-dl позволяет нам скачивать аудио только из видеороликов Youtube, например:

youtube-dl -x https://www.youtube.com/watch?v=iJvr0VPsn-s

По умолчанию Youtube-dl сохраняет аудио в формате Ogg (opus). Если вы хотите скачать аудио в любом другом формате, например в mp3, запустите:

youtube-dl -x --audio-format mp3 https://www.youtube.com/watch?v=iJvr0VPsn-s

Эта команда загрузит звук из данного видео / плейлиста, преобразует его в MP3 и сохранит в текущем каталоге.

Примечание. Для преобразования файлов в формат mp3 необходимо установить ffmpeg или avconv.

4. Загрузите видео с описанием, метаданными, комментариями, субтитрами и миниатюрами.

Чтобы загрузить видео и другую подробную информацию, такую ​​как описание, метаданные, комментарии, субтитры и эскизы, используйте следующую команду:

youtube-dl --write-description --write-info-json --write-annotations --write-sub --write-thumbnail https://www.youtube.com/watch?v=iJvr0VPsn-s

5. Перечислите все доступные аудио / видео форматы.

Видео и аудио на веб-сайте Youtube будут автоматически перекодированы в несколько аудио / видео форматов. Чтобы просмотреть все загружаемые аудио / видео форматы для видео или списка воспроизведения, используйте следующую команду:

youtube-dl --list-formats https://www.youtube.com/watch?v=iJvr0VPsn-s

Или аббревиатуру, которую обычно использует автор:

youtube-dl -F https://www.youtube.com/watch?v=iJvr0VPsn-s

Как показано на рисунке выше, Youtube-dl перечисляет все доступные форматы для данного видео слева направо:format code(Код формата видео),extension(имя расширения),resolution(Разрешение) иnote(Комментарий). Если вы хотите загрузить видео в определенном качестве или формате, очень удобно сначала проверить, что доступно.

6. Скачивайте видео в определенном качестве и / или формате.

По умолчанию Youtube-dl самостоятельно выбирает лучшее качество для скачивания видео. Однако также можно загружать видео или плейлисты в определенном качестве или формате.

Youtube-dl поддерживает следующие качества:

  • лучше всего выбрать аудио / видео файл лучшего качества
  • худший выберите формат худшего качества (видео и аудио)
  • bestvideo выбирает видеоформат наилучшего качества (например, DASH-видео), который может быть недоступен.
  • худшее видео Выберите чистый видеоформат с худшим качеством, которое может быть недоступно.
  • bestaudio выбирает аудиоформат самого высокого качества, который может быть недоступен.
  • худшийаудио Выберите аудиоформат с худшим качеством, который может быть недоступен.

Например, если вы хотите автоматически выбирать и загружать формат наилучшего качества (аудио и видео), просто используйте следующую команду:

youtube-dl -f best https://www.youtube.com/watch?v=iJvr0VPsn-s

Точно так же, чтобы загружать только аудио с лучшим качеством, выполните:

youtube-dl -f bestaudio https://www.youtube.com/watch?v=iJvr0VPsn-s

Вы также можете комбинировать следующие различные варианты формата:

youtube-dl -f bestvideo+bestaudio https://www.youtube.com/watch?v=iJvr0VPsn-s

Приведенная выше команда будетСкачать отдельноТолько видео самого высокого качествас участиемЧистый аудиоформат высочайшего качества, Затем используйте ffmpeg или avconv, чтобы объединить в файл MKV наилучшего качества; если вы не хотите объединять, замените + (плюс) на, (запятая), чтобы получить аудио и видео самого высокого качества (два файла), Следующим образом:

youtube-dl -f 'bestvideo,bestaudio' https://www.youtube.com/watch?v=iJvr0VPsn-s

7. Скачивание файлов с помощью видеокода (общий метод)

фронтМетод 5Как уже упоминалось, все видео Youtube имеют код формата, который мы можем использовать для загрузки видео определенного качества.

Например, используйтеМетод 5Просмотреть все доступные аудио / видео форматы и соответствующие имformat code(Код формата видео):

youtube-dl -F https://www.youtube.com/watch?v=iJvr0VPsn-s

Затем используйте код для загрузки указанного аудио / видео формата. Например, чтобы загрузить видеофайл наилучшего качества (код формата 22), выполните следующую команду:

youtube-dl -f 22 https://www.youtube.com/watch?v=iJvr0VPsn-s

При загрузке видео из плейлиста некоторые видео могут иметь другой формат. В этом случае можно указать несколько кодов формата в предпочтительном порядке, например:

youtube-dl -f 22/17/18 <playlist_url>

Согласно приведенному выше примеру Youtube-dl загрузит видео в формате 22 (если доступен); если формат 22 недоступен, он загрузит формат 17 (если доступен); если форматы 22 и 17 недоступны, наконец, попробуйте загрузить формат 18. . Если все коды формата не совпадают, Youtube-dl сообщит о подсказке. Также следует отметить, что косая черта является левоассоциативной, то есть крайний левый код формата является предпочтительным.

8. Загрузите аудио / видео через расширение файла.

Чтобы загрузить видео в предпочитаемом формате, например MP4, просто выполните:

youtube-dl --format mp4 https://www.youtube.com/watch?v=iJvr0VPsn-s

или

youtube-dl -f mp4 https://www.youtube.com/watch?v=iJvr0VPsn-s

Как я упоминал в предыдущем разделе, некоторые видео могут быть недоступны в предпочитаемом вами формате. В этом случае Youtube-dl загрузит другие наилучшие доступные форматы. Например, эта команда загрузит файл в формате MP4 самого высокого качества. Если формат MP4 недоступен, будет загружен другой наилучший доступный формат.

youtube-dl -f 'bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]/best' https://www.youtube.com/watch?v=iJvr0VPsn-s

9. Ограничьте размер загружаемых видео.

При загрузке нескольких видео из плейлиста Youtube вы можете захотеть загрузить только видео определенного размера. Например, эта команда не будет загружать видео меньше указанного размера, например 100 МБ:

youtube-dl --min-filesize 100M <playlist_url>

Если вы не хотите загружать видео больше заданного размера, вы можете сделать следующее:

youtube-dl --max-filesize 100M <playlist_url>

Мы также можем использовать комбинированные форматы и выбирать операторов для загрузки видео определенного размера. Например, следующая команда загрузит видео в лучшем видеоформате, но не более 100 МБ:

youtube-dl -f 'best[filesize<100M]' https://www.youtube.com/watch?v=iJvr0VPsn-s

10. Скачивание видео по дате

Youtube-dl позволяет нам фильтровать и загружать видео или плейлисты в зависимости от даты загрузки. Например, для загрузки видео, загруженных 1 августа 2019 года, вы можете использовать:

youtube-dl --date 20190801 <URL>

Загрузите видео, загруженные не ранее определенной даты:

youtube-dl --datebefore 20190801 <URL>

Скачать видео, загруженные не ранее определенной даты:

youtube-dl --dateafter 20190101 <URL>

Скачивайте только видео, загруженные за последние 6 месяцев:

youtube-dl --dateafter now-6months <URL>

Загрузите видео, загруженные в течение определенного периода времени (например, с 1 января 2018 г. по 1 января 2019 г.):

youtube-dl --dateafter 20180101 --datebefore 20190101 <URL>

11. Загрузите конкретное видео из плейлиста.

Загрузка определенных видео из плейлистов - еще одна очень полезная функция Youtube-dl. Например, чтобы загрузить 10-й файл из списка воспроизведения, используйте:

youtube-dl --playlist-items 10 <playlist_url>

Точно так же, чтобы загрузить несколько указанных файлов, просто разделите их запятыми:

youtube-dl --playlist-items 2,3,7,10 <playlist_url>

Конечно, вы также можете указать диапазон загрузок по порядковому номеру, например, начиная с 10-го, загрузить полный список напрямую:

youtube-dl --playlist-start 10 <playlist_url>

Или загрузите в плейлист только файлы с 2 по 5:

youtube-dl --playlist-start 2 --playlist-end 5 <playlist_url>

12. Скачивайте только видео, подходящие для определенного возраста.

Еще одна особенность Youtube-dl в том, что он позволяет скачивать только видео, подходящие для определенного возраста. Например, чтобы загрузить все элементы, не отмеченные как "NSFWВ видео "Или поиграем" только для 7-летних детей можно использовать:

youtube-dl --match-title "let's play" --age-limit 7 --reject-title "nsfw" <playlist_url>
